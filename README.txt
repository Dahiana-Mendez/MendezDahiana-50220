## Descripción del Proyecto
Proyecto Final es una aplicación web desarrollada en Django que permite a los usuarios registrarse y crear,
ver, editar y eliminar pacientes, citas médicas, historiales médicos y médicos.
Además, incluye una búsqueda de pacientes y médicos según ciertas pautas. 
El sistema de autenticación y autorización restringe el acceso a ciertas funciones a usuarios no registrados. 
Incluye también un sistema de edición de perfil y manejo de los avatares, como también un apartado de admin donde
se puede manejar toda la página.

La página de inicio tendrá el siguiente URL: http://127.0.0.1:8000/home
